# mermaid

美人鱼(mermaid)可以让你用文字和代码来表示图表。这简化了复杂图的维护。它是一个基于Javascript的图表和图表工具，可以呈现受markdown启发的文本定义来动态地创建和修改图表。美人鱼的主要目的是帮助文档跟上开发进度。
