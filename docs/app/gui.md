# wxFormBuilder

wxFormBuilder是一款非常好用的界面编辑设计工具，用于生成跨平台编译代码，能够快速搭建GUI开发环境，wxFormBuilder支持快速生成C++，python, Lua 等代码和 XRC 资源文件，搭建高效的带有GUI界面编辑器的开发环境。欢迎大家前来下载使用。

备注:
wxWidgets  发布库比较小
